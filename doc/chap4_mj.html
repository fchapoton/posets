‣ TheWallet( )( operation )

4.2 The Poset type

4.2-1 Ordering
‣ Ordering( X )( operation )

Returns: a function computing the order of the poset \(X\).

Ordering(X)(x,y) is the value of the expression \(x\geq y\).
